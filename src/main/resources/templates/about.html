About This Project

The Student Management System is a web-based application built with Spring Boot and Thymeleaf for efficient student data management. It enables educational institutions to maintain student records such as names, emails, and course details.

The project demonstrates CRUD (Create, Read, Update, Delete) functionality using Spring MVC architecture and Bootstrap for a modern responsive UI.
